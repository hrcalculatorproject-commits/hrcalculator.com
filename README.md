# Hunt Royale Advanced Build Planner

Advanced calculator for creating, comparing, saving, exporting, and sharing Hunt Royale builds.

This website is designed for players who want more detailed build planning than a basic set builder. It combines **Power Stones**, **Titan Gear**, and **Pets** into a single planner so you can experiment with different setups and see the full result in one place.

## What this tool does

* Build setups for **Helmet, Chest, Boots, Weapon, and Ring**
* Use **Raw, Super, and Mega Power Stones** in each available socket
* Plan **Titan Gear** with levels, enchants, and enchant ranks
* Select **1 Pet per build** and review its aura and ability
* View **Combined Build Stats** and **Overall Combined Stats**
* Track **Power Stone Gearscore**, **Titan Gear Score**, and **Overall Gear Score**
* Save unlimited loadouts locally in the browser
* Export builds and selected loadouts as shareable images

## Main features

### Build Planner

* Full build editor for all major equipment slots
* Socket-based Power Stone planning
* Support for Raw, Super, and Mega merges
* Quick socket copy/paste with buttons and keyboard shortcuts
* Easy mobile and desktop layout

### Titan Gear

* Titan Helmet, Armor, Boots, Weapon, and Ring support
* Level-based Titan stats
* Enchantment selection per item
* Enchant Rank selection per item
* Effective enchant rank calculation across equipped Titan pieces
* Titan enchant totals shown in the summary
* Bulk control to set all Titan levels at once

### Pets

* One pet can be selected per build
* Pets grouped by family type
* Displays pet ability and aura effect
* Pet data saved with each loadout
* Pet shown in exports

### Stats and Totals

* Defensive Power Stone stats
* Offensive Power Stone stats
* Titan enchant totals
* Overall combined stats
* Power Stone count summary
* Raw / Super / Mega usage count summary
* Saved loadout aggregate Power Stone counts

### Loadouts

* Unlimited saved loadouts
* Loadouts stored in local browser storage
* Save, load, rename, and delete builds
* Automatically creates a new blank loadout slot when needed
* Choose which loadouts to include in export

### Exporting

* Export the current build as an image
* Export selected loadouts as one long scrollable image
* Export includes:

  * equipped Power Stones
  * Titan levels
  * Titan enchants and ranks
  * pet information
  * gear score totals
  * combined stat sections

## Keyboard shortcuts

* **Ctrl + C / Cmd + C** — Copy selected socket
* **Ctrl + V / Cmd + V** — Paste selected socket
